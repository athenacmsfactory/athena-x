# 🎨 Guide for Generating Images for Your Sites

The Unsplash links generated by AI are sometimes unreliable (404 errors). This guide helps you generate high-quality images **yourself** using AI tools like **Google Antigravity Image Generator** or **DALL-E**.

## 📋 How Does It Work?

### Step 1: Identify Which Images You Need

Look in the `src/data/` directory of your site (e.g., `the-cheese-house`). Each JSON file contains items that require an image.

**Example:** `customer_interaction.json`
```json
{
  "title": "Absolutely the best Polder cheese!",
  "customer_name": "Suzanne W.",
  "interaction_type": "Review"
}
```

### Step 2: Generate a Suitable Prompt

For each item, use this formula:

```
Professional product photography of [SUBJECT], [STYLE], [ATMOSPHERE], high quality, 4k
```

**Examples per Niche:**

### Step 3: Generate the Image

1. **Open Google Antigravity Image Generator** or another AI image tool.
2. Paste the prompt.
3. Generate the image.
4. Download as `.jpg` or `.webp`.

### Step 4: Place the Image in Your Site

1. **Save the image** in: `sites/[SITE-NAME]/public/images/`
2. **Give it a clear name**, for example: `cheese-polderkroon.jpg` or `review-suzanne.jpg`
3. **Update the JSON file:**

```json
{
  "title": "Absolutely the best Polder cheese!",
  "image_url": "cheese-polderkroon.jpg"  // Relative path, NO leading slash
}
```

**Note:** Do **NOT** use a leading slash (`/`). The code automatically appends `BASE_URL + 'images/'`.

## 🔧 Troubleshooting

### "Image Not Loading"
- Check if the file is actually in `public/images/`.
- Check if the filename matches **exactly** (case-sensitive!).
- Do **not** use spaces in filenames (use `-` or `_`).

### "I Still See placeholder.jpg"
- Refresh the page with `Ctrl+F5` (hard refresh).
- Check if the JSON file was saved correctly.
